Array.prototype.filter()
        filter() 메서드는 조건 결과를 새로운 배열로 반환합니다.
        
        arr.filter(function(현재값) => {
            return 조건
        })
        arr.filter( 현재값 => 조건 )

        매개변수
        callback
        각 요소를 시험할 함수. true를 반환하면 요소를 유지하고, false를 반환하면 버립니다. 다음 세 가지 매개변수를 받습니다.
        element
        처리할 현재 요소.
        index Optional
        처리할 현재 요소의 인덱스.
        array Optional
        filter를 호출한 배열.
        thisArg Optional
        callback을 실행할 때 this로 사용하는 값.
        반환 값
        테스트를 통과한 요소로 이루어진 새로운 배열. 어떤 요소도 테스트를 통과하지 못했으면 빈 배열을 반환합니다.
